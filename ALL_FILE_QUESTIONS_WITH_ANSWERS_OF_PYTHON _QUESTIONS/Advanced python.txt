Metaclass in Python
✅ Definition

A metaclass is:

A class that defines how other classes are created.

👉 In short:

Object → created by Class  
Class → created by Metaclass

Default metaclass = type

🧠 Real-time Use Cases
Enforcing coding standards
Auto-registering classes (Django ORM, serializers)
Validation frameworks
Singleton enforcement




class UpperAttrMeta(type):
    def __new__(cls, name, bases, dct):
        new_dct = {}
        for key, value in dct.items():
            if not key.startswith("__"):
                new_dct[key.upper()] = value
            else:
                new_dct[key] = value
        return super().__new__(cls, name, bases, new_dct)


class Demo(metaclass=UpperAttrMeta):
    name = "Abhi"

print(hasattr(Demo, "NAME"))  # True

30. What is Thread Safety?
✅ Definition

Thread safety ensures:

Multiple threads can access shared data without corruption



import threading

counter = 0

def increment():
    global counter
    for _ in range(100000):
        counter += 1

threads = [threading.Thread(target=increment) for _ in range(2)]

for t in threads:
    t.start()
for t in threads:
    t.join()

print(counter)  # unpredictable



lock = threading.Lock()

def increment():
    global counter
    for _ in range(100000):
        with lock:
            counter += 1

43. Closure in Python
✅ Definition

A closure is:

A function that remembers values from its enclosing scope even after the outer function has finished execution.

🧠 Real-time Use Cases
Decorators
Function factories
Data hiding


def outer(x):
    def inner(y):
        return x + y
    return inner

add5 = outer(5)

print(add5(3))   # 8




def counter():
    count = 0

    def increment():
        nonlocal count
        count += 1
        return count

    return increment

c = counter()
print(c())  # 1
print(c())  # 2
